Timer.js

by CJ Cruz

Timer.js provides easy-to-use Javascript timer which uses real-time concurrent timers and allows different actions to be done on completion.

There are two required functions with sensitive names for this to work:
function start(id) - actions when id is started (preferrably switch-case)
function complete(id) - actions when id is completed (preferrably switch-case)
The timer.js should be added after these two functions.

Another function can be created to get values from text inputs, similar to function start_timer(id)
